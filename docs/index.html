Everything you need remotely

A lightweight dashboard for developers who leave long sessions running and want to stay connected.

Live Terminal

Full xterm.js rendering in the browser. See real-time output and send input from your phone as if you were at your desk.

Git Diffs

Review staged and unstaged changes with syntax-highlighted diffs. Stage files, unstage them, and commit — all from the dashboard.

File Browser

Navigate your project tree and read any file directly in the browser. No extra setup, gitignore-aware.

Multi-session

Run multiple terminal sessions from one daemon. Switch between them in the dashboard — each one persists independently.

Mobile-first

Designed for phones first. Clean dark UI, large touch targets, and a virtual keyboard for sending commands on the go.

Zero dependencies

Single binary with Node.js bundled. No tmux, no Docker, no cloud — just one install command and it runs on your machine.

Up and running in 30 seconds

Install once, use from any device on your local network

1

Install

Run the install script. devpilot unpacks to ~/.devpilot with a bundled Node.js — no system Node required.

2

Start a session

Run devpilot start (or devpilot start claude). The dashboard URL is printed immediately.

3

Open on your phone

Visit the URL on any device on the same network. Live terminal, diffs, and files are a tap away.
